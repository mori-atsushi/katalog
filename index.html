Katalog

A UI Catalog Library made with Jetpack Compose

Getting Started

Why using the UI Catalog?â€‹

When implementing complex UIs, you need to be aware of various things such as omissions and degradation.

The UI Catalog can be build UIs separately from the logic, so it is easy to reproduce hard-to-reach states and edge cases. It will be useful for reuse UIs by organizing and managing UI components.

The UI Catalog will powerfully help for your development.

Features of Katalogâ€‹

It is made with Jetpack Compose and you can preview Composable. You can easily register and group UI components using the DSL , as shown below.

class MyApplication : Application() {
    override fun onCreate() {
        super.onCreate()

        registerKatalog(
            title = "My App Catalog"
        ) {
            compose("UI Component") {
                Text(text = "Hello, World")
            }
        }
    }
}

In addition to Composable, the following UI components are supported

  • Android View
  • DataBinding / ViewBinding
  • Fragment
  • Android View
  • DataBinding / ViewBinding
  • Fragment
view(
    name = "TextView"
) {
    TextView(context).apply {
        text = "Hello, World"
    }
}
binding(
    factory = SampleBinding::inflate,
    name = "Binding Sample"
)
fragment {
    SampleFragment()
}
